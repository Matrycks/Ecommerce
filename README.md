# Ecommerce Management Application

A demo ASP.NET Core 9 application built with Clean Architecture for managing orders, products and customers.

![.NET](https://img.shields.io/badge/.NET-9.0-blue)
![Build](https://github.com/Matrycks/Ecommerce/actions/workflows/build.yml/badge.svg)

## Features

- Order processing and tracking

## Architecture

- Clean Architecture with dependency inversion

## Tech Stack

- ASP.NET Core 9
- EF Core 9
- SQL Server

## Getting Started

### Prerequisites

- .NET SDK 9.0
- SQL Server / Docker

### Setup

```bash
git clone https://github.com/Matrycks/Ecommerce.git
cd repo
dotnet restore
dotnet run --project src/Web
```
